--Prerequisites--
1. You will need a Google API key to use this application. If you do not have one, it can be obtained here:
    https://developers.google.com/maps/gmp-get-started

2. Press the button that says "Get Started" and follow the steps

3. Make sure your API key is ENABLED for use with the Google Directions API 


--Once you have completed the Prerequisites, follow the steps below to run the application.--


--Step 1--
Clone this repository using the following command:
    'git clone https://github.com/m-maka/CIS-422-Project1.git'

--Step 2--
Change directories so you are in the 'src' folder:
    'cd src'

--Step 3--
Input your API key in settings.py:
    Open the 'settings.py' file and replace the <INSERT API KEY> inside the quotes with your own API key 

--Step 4--
Install dependecies for the program from the requirements.txt using the following command:
    'pip install -r requirements.txt'

--Step 5--
Activate your environment using the following command:
    'venv\Scripts\activate'

--Step 6--
Start the application by running start.sh:
    'sh start.sh'
